# Gitar Öğreniyorum

## Overview

Modern, playful guitar learning web app for Turkish middle school students (ages 10-14). Built as a pnpm monorepo.

## Stack

- React + Vite frontend (`artifacts/guitar-app`)
- Express 5 + Drizzle ORM API (`artifacts/api-server`)
- PostgreSQL (Replit-managed)
- JWT auth (1h expiry) using `SESSION_SECRET`
- OpenAPI-first contract via Orval (`lib/api-spec`, `lib/api-client-react`, `lib/api-zod`)
- Web Audio API for note playback, Framer Motion for animations

## Roles

- **Admin** — password `114344_Kenan` (modal on landing page, opens via animated bass clef in top-right). Manages institutions and edits per-institution teacher/student limits.
- **Teacher** — 2-step login: enters invite code, sees institution name, then submits first/last name + optional "Beni Hatırla" toggle. Saved credentials enable one-click quick login next visit. On class creation, specifies a `studentCount` and N **8-digit numeric** student codes are pre-generated. Class capacity can be expanded later. Each class also has a **6-digit smartboard code** (generated on demand) that opens a public `/smartboard/:code` classroom view with no level restrictions.
- **Student** — logs in with an invite code generated by a teacher (single name field).

## Quota model

- Each institution has `teacherLimit` and `studentLimit` (admin-editable; cannot be set below current usage).
- `usedTeacherCount` = number of generated teacher codes; `usedStudentCount` = SUM of `classes.studentCapacity`.
- Creating or expanding a class consumes student quota immediately (codes pre-generated).

## Routing note

- All sub-routers in `artifacts/api-server/src/routes/{admin,teacher,student}.ts` use **path-prefixed `requireAuth` middleware** (`router.use("/admin", requireAuth(["admin"]))`) — NOT `router.use(requireAuth([...]))`. Without the path prefix, Express runs the middleware for every `/api/*` request even if the route belongs to a different sub-router.

## Lessons

- Module 1: Gitar Tutuş, Akort + Notlar (interactive fretboard), Sağ El Ritim
- Module 2: Em Akoru, Am Akoru (locked until teacher unlocks level 2)

## Smartboard Mode

- Public route: `/smartboard/:code` (6-digit numeric code)
- Teacher generates code from dashboard → "Akıllı Tahta Kodu Oluştur" button on each class card
- Smartboard page shows all lessons with NO level lock — teacher can demonstrate any lesson in class
- API endpoint: `GET /api/smartboard/:code` (no auth required)
