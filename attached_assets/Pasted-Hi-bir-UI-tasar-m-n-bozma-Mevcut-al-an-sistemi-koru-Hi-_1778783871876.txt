ŞİMDİ YAPILACAKLAR:

1. ADMIN_PASSWORD güvenliği:
Kod içinde:
process.env["ADMIN_PASSWORD"] ?? "114344_Kenan"
veya benzeri fallback varsa kaldır.

ADMIN_PASSWORD environment variable yoksa:
server açık hata mesajı verip başlamasın.

Admin şifresi kaynak kodda hiçbir yerde hardcoded kalmasın.

2. Render sleep + RAM cache güvenliği:
Şu an “Bunu öğrendim” sistemi RAM cache kullanıyorsa:
- veri kaybı yaşanmayacak şekilde güvenli hale getir.
- lesson_progress anında DB’ye yazılmaya devam etsin.
- öğrenci ilerlemesi hiçbir durumda kaybolmasın.
- 120 saniyelik performans cache mantığı korunabilir.
- Ancak Render sleep/wake durumunda pending learning request tamamen kaybolmasın.

Mümkünse:
- RAM cache sadece performans amaçlı kullanılsın.
- Kritik öğrenme isteği DB’ye güvenli şekilde yazılsın.
- Flush başarısız olursa retry sistemi devam etsin.
